Before Getting Started
---------------------

Delete the existing CertifiedOverheadCrane DB.
Delete the contents of the App_Data folder (dependencies doesn't matter, though).
Copy the contents of Orchard.Web to the root directory.
Setup DB permissions using IIS-DatabaseUser.sql
Grant Access Control *Modify* permissions for the IIS Apppool for these folders.
	App_Data
	Media
	Themes	
	Modules
